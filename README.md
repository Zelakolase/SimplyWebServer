# SimplyWebServer
A developing project for web server made in java. All commits after this update go under Attribution-NonCommercial-ShareAlike 4.0 International license. [More about this license](https://creativecommons.org/licenses/by-nc-sa/4.0/).<br>
## Choose your version.
- SimplyWebServer (SWS) : Dynamic mode is java code. Extend Server object and code your app in the abstract function (we call it: dynamic engine).
- SimplyWebServer Hybrid (SWS-Hybrid) : Dynamic mode can be done with **ANY** programming language. Latency issues may occur, to be fixed. (9Feb. 2022).
